# Projet DevOps – Gestion de Bibliothèque

## Description

Ce projet est réalisé dans le cadre de l'examen de DevOps du Dakar Institute of Technology (DIT).

L'objectif est de concevoir, développer et déployer une application web moderne de gestion de bibliothèque basée sur une architecture microservices.

L'application permettra notamment de :

- Gérer les livres
- Gérer les utilisateurs
- Gérer les emprunts
- Automatiser le déploiement avec Docker et Jenkins

---

## Technologies utilisées

### Backend
- FastAPI

### Frontend
- HTML
- CSS
- JavaScript

### Base de données
- PostgreSQL

### Conteneurisation
- Docker
- Docker Compose

### Intégration Continue / Déploiement Continu
- Jenkins
- GitHub

---

## Fonctionnalités

### Gestion des livres
- Ajouter un livre
- Modifier un livre
- Supprimer un livre
- Rechercher un livre

### Gestion des utilisateurs
- Ajouter un utilisateur
- Consulter les utilisateurs
- Gérer les profils

### Gestion des emprunts
- Emprunter un livre
- Retourner un livre
- Consulter l'historique
- Détecter les retards
